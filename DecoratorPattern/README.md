## Comparison

`Strategy pattern`是内核的修改、替换，`decorator pattern`是保留内核，在外层进行复式装修

`Template method`着重保留公共functions，并定义统一的流程，`decorator pattern`保留内核的同时，侧重于n种装饰的组合套用
